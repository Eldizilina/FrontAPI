Introdução

Esta API fornece acesso a informações sobre pontos turísticos de são tomé organizados por distritos. Todos os endpoints retornam dados no formato JSON.

Autenticação: Atualmente não é necessária autenticação para acessar os endpoints públicos.

Listar Distritos

GET /distritos

Retorna a lista de todos os distrito que se encontram disponiveis na base de dados e a quantidade de pontos turisticos existentes neles.

Exemplo de Requisição
fetch('https://api-webservices-1.onrender.com/api/distritos') .then(response => response.json()) .then(data => console.log(data));
Resposta
{ [ { "nome": "Agua grande", "quantidade": 5 }, { "nome": "Cantagalo", "quantidade": 5 }, { "nome": "Caue", "quantidade": 10 }, { "nome": "Lemba", "quantidade": 3 }, { "nome": "Lobata", "quantidade": 3 }, { "nome": "Me-zochi", "quantidade": 3 }, { "nome": "Rap", "quantidade": 8 } ] }

Pontos por Distrito

GET /pontos-turisticos/distrito/{nome}

Retorna todos os pontos turísticos de um distrito específico.

Parâmetros
Nome distrito Localização Descrição Cidade Pontos Proximos Imagem URL
Exemplo de Requisição
fetch('https://api-webservices-1.onrender.com/api/pontos-turisticos/distrito/lobata') .then(response => response.json()) .then(data => console.log(data));
